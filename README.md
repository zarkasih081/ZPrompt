# Zprompt

Website statis untuk membuat prompt desain AI secara cepat dan rapi.

## Isi file

- `index.html` halaman utama
- `style.css` tampilan website
- `script.js` logika generate prompt, copy, download TXT, mode gelap
- `assets/logo.svg` logo Zprompt
- `assets/favicon.svg` favicon website
- `manifest.webmanifest` konfigurasi PWA dasar

## Cara pakai lokal

1. Ekstrak ZIP.
2. Buka `index.html` di browser.
3. Isi form.
4. Klik Generate Prompt.
5. Klik Copy Prompt atau Download TXT.

## Cara publish ke GitHub Pages

1. Buat repository baru di GitHub.
2. Upload semua file di folder ini.
3. Buka Settings.
4. Pilih Pages.
5. Source: Deploy from a branch.
6. Branch: main, folder root.
7. Simpan.

## Cara publish ke Netlify

1. Login Netlify.
2. Pilih Add new site.
3. Pilih Deploy manually.
4. Drag folder hasil ekstrak ke Netlify.
5. Website langsung aktif.
